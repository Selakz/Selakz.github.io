TAKANA³更新日志

  • 2025-09-19
  • 作者 Senolytics
  • 844 字

0.3.4 更新内容

  • 添加了切换为全屏模式的功能
  • 添加了调整相机视角的功能
  • 添加了位置指示器（可在设置中关闭）
  • 添加了选中两个 Note 后按 H 键将其连接为 Hold 的功能

0.3.3 更新内容

  • 提供了制谱器内的设置调整面板
  • 添加了自动检测更新并下载安装的功能

0.3.2 更新内容

  • 修复切换谱面时前一个谱面的状态未完全清除的问题
  • 修复滚轮卡死的问题
  • 修复曲绘无法加载的问题
  • 修复可以选中已经消失的 Note 的问题
  • 修复多选轨道时只镜像一个轨道的问题
  • 修复复制物件时编辑器信息未复制的问题
  • 修复部分情况下 Hold 不显示的问题
  • 修复部分 UI 显示问题

0.3.1 更新内容

  • 进行重构，采用了新的项目结构与谱面格式，且性能、观感上均有所提升
  • 对图层功能进行改善，支持轨道透明度、轨道染色、装饰图层等功能
  • 添加了设置假 Note 的功能
  • 添加了在编辑面板中为元件起名，并根据 ID 或名称进行搜索的功能
  • 多选轨道时，多个轨道均可以显示运动轨迹
  • 添加了通过命令行参数直接加载项目的功能

0.2.2 更新内容

  • 添加了打击音
  • 添加了镜像以及镜像复制的功能
  • 添加了设置默认曲线族的功能
  • 轨道信息在只选中一条轨道时将占满整个编辑面板
  • 部分 bug 修复和 UI 改动

0.2.1 更新内容

  • 添加了修改窗口分辨率的功能
  • 添加了更多类型的缓动曲线
  • 将切换创建 Note 类型的快捷键调整为 ` 键

0.2.0 更新内容

  • 添加了按住 Ctrl 进行多选的功能
  • 添加了轨道图层功能
  • 添加了复制粘贴功能
  • 添加了捕获异常并输出日志的功能

0.1.1 更新内容

  • 添加了修改初始轨道时间⻓度的设置项
  • 添加了左键暂停播放的设置项
  • 添加了用 Space 键切换播放状态的功能
  • 修改了格线颜色
  • 修改了 Note 皮肤
  • 现在运动曲线以动态精度进⾏渲染
  • 现在谱面导出时内容根据时间排序
  • 修复了谱面编辑与导出时的一些问题
  • 修复了取消选中时编辑面板中的内容不消失的问题
  • 修复了其他的一些格式问题
